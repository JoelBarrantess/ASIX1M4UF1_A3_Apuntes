## Primer Capitulo: MARKDOWN

Como poner el texto en *cursiva*?

```
*texto* o _texto_
```

Como poner el texto en **Negrita?**

```
**Texto** o __Texto__
```

Como Juntar ***las dos?***

```
Hay varias maneras, pero basicamente es alterar entre los dos, por ej:
_**texto**_ o __*texto*__...

```
